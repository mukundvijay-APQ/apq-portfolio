GD&T: Tolerance Stack-Up for a Multi-Sensor Ring Across 12 Sizes

Mukund Vijay · Fictional scenario

Situation

A health-monitoring ring contained 6 sensor components on a flexible PCB curving to match the ring's inner surface. Ring sizes 4–15 meant the flex PCB curvature radius ranged from 7.5mm to 11.2mm. Challenge: ensuring sensor-to-skin contact distance within ±0.15mm across all sizes, despite changing geometry.

GD&T Framework

Datum Structure

Datum A: Inner ring surface (skin-contact face) — primary functional reference

Datum B: Ring centerline axis — controls rotational position of sensors

Datum C: Sensor window center — locates optical path relative to Datum A

TOLERANCE STACK ANALYSIS

6-component stack: Inner shell ±0.05mm, adhesive ±0.03mm, flex PCB ±0.02mm, sensor dome ±0.04mm, epoxy ±0.05mm, curvature offset ±0.03mm

Worst-case: ±0.22mm — exceeds ±0.15mm requirement

RSS (3σ): ±0.09mm — within requirement with margin

Monte Carlo (100K iterations): 99.7% within ±0.13mm

Size-Specific Compensation

Created size-specific epoxy dispensing profiles:

  • Sizes 4–6: 0.30mm (standard)
  • Sizes 7–9: 0.34mm (+0.04mm compensation)
  • Sizes 10–12: 0.39mm (+0.09mm compensation)
  • Sizes 13–15: 0.44mm (+0.14mm compensation)

Results

METRICBEFOREAFTER
Sensor-to-skin gap consistency±0.28mm±0.11mm

Key Takeaways

The original drawings used basic ± tolerances referenced to manufacturing datums — which had no relationship to the functional requirement of sensor-to-skin distance. Redefining the datum structure around functional requirements aligned the tolerance scheme with what actually mattered. The size-specific epoxy compensation table was the critical insight.
